## 2. Среди за програмиране
- ***Idle***: <br>
         - pros: научаване на синтаксиса на езика<br>
         - cons: няма подсказки, по- трудна работа с пакети
- ***IDE(PyCharm)***:<br>
         - pros: има подсказки, лесна работа с пакети<br>
         - cons: по- трудно запомняне на вградените функции
